
1) Add launcher shortcut for terminal;

2) terminal keyboard preferences: check "use option as meta key"

3) Map caps lock to control on all keyboards;
   SystemPreferences->Keyboard[Keyboard]->ModifierKeys; for all keyboards, set the caps lock mapping;

4) Download/install/link chrome;

5) Create/install ssh keys;

6) turn on click-and-drag -
   System Preferences ->
     Accessibility ->
       Mouse & Trackpad from options on left ->
         click Trackpad Options ->
           place checkmark next to "Enable dragging"

7) install brew:
   /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

8) brew install awscli

9) install emacs;
   see: https://github.com/railwaycat/homebrew-emacsmacport
   (from hackernews https://news.ycombinator.com/item?id=9594031)

   brew tap railwaycat/emacsmacport
   brew install emacs-mac
   brew linkapps

10) Keyboard shortcuts for Terminal app(Utilities/Terminal), Chrome, Emacs;
    see: https://computers.tutsplus.com/tutorials/how-to-launch-any-app-with-a-keyboard-shortcut--mac-31463

11) Setup python3 environment:
    brew install python3
    pip3 install virtualenv
    pip3 install pylint

12) Other stuff;
    brew install jq
